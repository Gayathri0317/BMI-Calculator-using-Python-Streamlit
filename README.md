# BMI-Calculator-using-Python-Streamlit
A Simple and Attractive BMI Calculator built using Python &amp; Streamlit 

# ⚖️ BMI Calculator with Chart
## 🚀 Features
- Calculates BMI
- Shows BMI category
- Displays colorful BMI chart
- Attractive UI design

## 🛠️ Technologies Used
- Python
- Streamlit
- Matplotlib

## ▶️ How to Run
pip install -r requirements.txt
streamlit run Bmi_calculator.py
